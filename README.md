# EnvMon

EnvMon stands vor Environment Monitoring.

This project is a demonstrator for a serverless approach to monitoring environment data.

## Technologies

### Sensors

* Raspberry PIs
* DHT11 sensors
* AWS IoT SDK for node.js

### Backend

* AWS IoT
* AWS Cognito
* AWS DynamoDB
* AWS S3

### Website

* Facebook authentication
* AWS SDK
* Plot.ly
